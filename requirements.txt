###  Why these
- `python>=3.10` → Ensures use a modern Python version.  
- `pytest>=7.0` → Required to run your `test_game.py` suite.  

---

###  How to use
After cloning your repo, anyone can install dependencies with:
```bash
pip install -r requirements.txt
```

Then run the game:
```bash
python main.py
```

Or run the tests:
```bash
pytest -v
```
